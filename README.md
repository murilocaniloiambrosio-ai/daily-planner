## About the Project

**Daily Planner** is a productivity-focused daily planning application built around the principle that the best tool is the one that gets out of your way.

A clean, minimal UI with **persistent state management** that keeps your tasks across sessions — no account required, no tracking, no friction.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React · Vite |
| Language | JavaScript |
| Persistence | LocalStorage |
| Deployment | GitHub Pages |

---

## Features

- **Add, edit, and complete tasks** with instant UI feedback
- **Persistent storage** — your data survives page reloads
- **Minimal, distraction-free interface**
- **Mobile-friendly responsive layout**
- **Zero account required** — runs entirely client-side

---

## Engineering Highlights

- **Pure client-side architecture** — no backend, no tracking
- **Component-driven React** with clean state management
- **LocalStorage persistence** with safe serialization
- **Fast Vite build** producing a small production bundle
- **Production deployment** via GitHub Pages CI

---

## Local Setup

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

No environment variables required.

---

## Security & Privacy

- **No data leaves your device** — all storage is local browser storage
- **No analytics, no tracking, no third-party scripts**
- No secrets or credentials present in the repository
- `.gitignore` excludes build artifacts and local files
